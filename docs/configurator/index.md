# Configurator

This section documents the WingFlight Configurator tab by tab, in the order
they appear once connected to a flight controller.

The Configurator's tab list adapts to your connection state and firmware
capabilities -- some tabs (like ESC Programming or GPS) only appear if the
relevant feature is enabled or detected.

| Tab | Purpose |
|---|---|
| [Status](tabs/status.md) | Live connection/system status |
| [Setup](tabs/setup.md) | 3D orientation preview, sensor status |
| [Configuration](tabs/configuration.md) | Board alignment, receiver mode, ESC/motor protocol, system-level settings |
| [Power](tabs/power.md) | Battery/voltage/current monitoring |
| [Receiver](tabs/receiver.md) | RX protocol and channel mapping |
| [Failsafe](tabs/failsafe.md) | Failsafe stage configuration |
| [Mixer](tabs/mixer.md) | Airframe/control-surface mixing |
| [Servos](tabs/servos.md) | Per-servo output configuration |
| [Motors](tabs/motors.md) | Motor output testing and configuration |
| [Curves](tabs/curves.md) | Throttle and control curves |
| [Profiles](tabs/profiles.md) | PID tuning profiles |
| [Rates](tabs/rates.md) | Stick response/rate profiles |
| [Gyro](tabs/gyro.md) | Gyro filtering |
| [Auxiliary (Modes)](tabs/auxiliary.md) | Flight-mode-to-switch mapping |
| [Adjustments](tabs/adjustments.md) | In-flight tuning via transmitter switches |
| [Logic Conditions](tabs/logic.md) | Custom in-flight logic |
| [GPS](tabs/gps.md) | GPS setup and rescue |
| [LED Strip](tabs/led-strip.md) | Addressable LED configuration |
| [Beepers](tabs/beepers.md) | Audible alert configuration |
| [Sensors](tabs/sensors.md) | Raw sensor data |
| [ESC Programming](tabs/esc-programming.md) | Passthrough ESC configuration |
| [Blackbox](tabs/blackbox.md) | Flight-data logging |
| [CLI](tabs/cli.md) | Direct command-line access |
