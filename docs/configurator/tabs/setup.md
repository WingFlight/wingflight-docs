# Setup

The Setup tab shows a live 3D model of the aircraft that mirrors the flight
controller's reported attitude in real time, alongside sensor status
indicators (gyro, accelerometer, magnetometer, barometer, GPS).

Use this tab to:

- Confirm the board orientation setting matches how the flight controller is
  physically mounted -- the 3D model should roll/pitch the same direction as
  the real aircraft.
- Spot-check that all fitted sensors are detected and reporting.
