# GPS Rescue

GPS Rescue automatically returns the aircraft toward its arm location (or a
configured rescue point) when triggered, typically as a [Failsafe](../configurator/tabs/failsafe.md)
response to lost radio link, provided a GPS fix is available.

GPS Rescue requires a working [GPS](../configurator/tabs/gps.md) fix before
it can be relied on -- test and tune rescue behavior deliberately (in open
space, with props at a safe altitude and heading) before trusting it as your
failsafe plan.
